### Installation

##### Just, Clone this repository -
```sh
$ https://github.com/ImUpXD/ImUpPhisher.git
```

##### Change to cloned directory -
```sh
$ cd phisher
$ chmod 777 phisher.sh
$ bash phisher.sh
```
